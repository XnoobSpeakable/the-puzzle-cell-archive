Everything that isn't in brackets is the official version of the game (without codename, just version)
Now I'll explain the cryptic terms that are in brackets

UT means "Unopened: true"
UU means "Unopened: unknown"
When you open the game for the first time, it generates some settings, so an unopened copy is different from the opened one. The ons labaled UT haven't been opened.

OK means the game works and is the same version as the name of the folder

And finally, the last number means how early the version was released. A version with a lower number was released before a version with a bigger number.
